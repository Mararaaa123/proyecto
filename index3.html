¿Qué es un recurso frontend y cuál es su función en el desarrollo web?

El frontend se dedica a definir los procesos de interacción con el usuario. Pero muchos de estos procesos implican la comunicación con el servidor para acceder a las bases de datos, iniciar sesión en la aplicación, autorizar determinadas acciones o realizar pagos electrónicos, entre otras muchas cosas.

¿Cuáles son las principales tecnologías o herramientas utilizadas en el desarrollo frontend y por qué son importantes?

1. HTML HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el núcleo del desarrollo frontend. Gracias a esta tecnología, se puede crear la estructura de un sitio web. En los elementos HTML se puede brindar opciones para agregar imágenes, formularios, contenedores y mucho más.
2. CSS Otra tecnología frontend fundamental es CSS (Cascading Style Sheets o Hojas de Estilo en Cascada), esta tecnología ayuda a crear algunas reglas de estilo adicionales a la estructura HTML. Ademas esta tecnología permite que una aplicación web sea responsiva (que se vea bien en todos los tamaños de las pantallas) y sea mas interactiva con algunas animaciones, gracias a esta tecnología.
JavaScript es un lenguaje de programación que te permite implementar funciones complejas en páginas web. De modo que cualquier característica que este del lado del frontend puede utilizar JavaScript. Ya sean actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, etc., JavaScript está involucrado.

Describe el concepto de responsividad en el diseño web y menciona al menos dos técnicas para lograrlo.

El Responsive Design o diseño adaptativo es la técnica que se usa en la actualidad para tener una misma web adaptada a las diferentes plataformas que nos brinda la tecnología: ordenador, tablet, Smartphone, iPad, Kindle y, en resumen, todas las resoluciones de pantalla existentes.

¿Cuál es la diferencia entre HTML, CSS y JavaScript en el contexto del desarrollo frontend? Explique brevemente el papel de cada uno de estos lenguajes.

HTML (Hyper Text Markup Language): HTML es un lenguaje de marcado utilizado para estructurar el contenido de una página web.este lenguaje define la estructura de la página, como encabezados, párrafos, listas, enlaces y otros elementos. comunmente sus etiquetas son por ejemplo h1 div nav, etc. Es la base sobre la cual se construye toda la página web.
CSS (Cascading Style Sheets): CSS es un lenguaje utilizado para dar estilo a los elementos HTML. Esto controla la apariencia visual de la página, como colores, fuentes, márgenes y diseños.una de sus etiquetas son por ejemplo p { color: blue; } cambia el color del texto de los párrafos a azul. Mejora la presentación y la coherencia visual de la página.
JavaScript: JavaScript es el único lenguaje de programación de los tres. Agrega interactividad y funcionalidad dinámica a las páginas web. document.getElementById('miBoton').addEventListener('click', function() { alert('¡Hola!'); }); crea una alerta cuando se hace clic en un botón. Permite crear aplicaciones web interactivas y dinámicas.
